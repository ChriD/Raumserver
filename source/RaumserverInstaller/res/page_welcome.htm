Welcome to the Raumserver-Installer

This installer helps you to install the Raumserver application onto a Raumfeld device

For this the favored Raumfeld device has to have SSH enabled
This can be easily done following this instructions

  1. Get the SSH activation file located in the \enableSSH directory of the installer
  2. Copy this file to the root directory of a FAT32 formated USB Stick
  3. Put the USB stick into the usb hub of the favored raumfeld device
  4. Wait about a minute, then remove the USB Stick
  5. Voila! Now you have enabled SSH access to this device!
For more information visit: Raumserver on Github
